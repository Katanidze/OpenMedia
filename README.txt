                                                      Инструкция запуска

Для запуска программы требуется в консоли открыть файл Project.exe. Файл Project.exe находится по пути \x64\Debug\Project.exe.
Внимание. Открытие файла Project.exe вне папки \x64\Debug приведет к тому, что динамическая библиотека подключена не будет. Для корректной работы
требуется переместить также и .dll файлы.

                                                      Инструкция открытия проекта в программе Visual Studio 2022

Для открытия проекта в программе и изменения кода следует открыть файл OpenMedia.sln.
